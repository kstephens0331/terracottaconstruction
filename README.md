# Terracotta Construction

## Overview

Monorepo for the complete Terracotta Construction digital platform. This project encompasses the public-facing SEO marketing website, an internal admin portal, a customer-facing project portal, and the backend API server — all architected as separate applications under a unified multi-subdomain deployment strategy.

## Tech Stack

| Application | Stack |
|---|---|
| **SEO Website** () | Next.js, React, Tailwind CSS |
| **Admin Portal** () | React, Vite, Tailwind CSS |
| **Customer Portal** () | React, Vite, Tailwind CSS |
| **API Server** () | Node.js, Express |
| **Deployment** | Vercel (multi-subdomain) |

## Features

### SEO Marketing Site
- **Search Engine Optimization** — Server-side rendered pages with structured data, meta tags, and sitemap generation
- **Performance Optimized** — Next.js static generation and image optimization for fast page loads
- **Service Showcases** — Detailed pages for each construction service offering
- **Lead Generation** — Contact forms and call-to-action components throughout the site

### Admin Portal
- **Project Management** — Create, update, and track construction projects through their lifecycle
- **Client Management** — Maintain customer records, contracts, and communication history
- **Team Coordination** — Assign tasks, manage schedules, and track workforce allocation
- **Financial Overview** — Revenue dashboards, invoice tracking, and payment status

### Customer Portal
- **Project Visibility** — Customers can view real-time progress on their construction projects
- **Document Access** — Download contracts, invoices, permits, and project documentation
- **Communication** — Direct messaging with the Terracotta Construction team
- **Payment Management** — View payment schedules and transaction history

### Branding & Assets
- **Consistent Identity** — Shared design tokens, color palettes, and typography across all applications
- **Asset Library** — Centralized logos, icons, and brand imagery
